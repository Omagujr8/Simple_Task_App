# 📝 Simple Task Manager App

A beginner-friendly JavaScript project that helps users manage daily tasks directly in the browser. This project focuses on core frontend fundamentals without any frameworks like React.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 💾 Persistent storage using `localStorage`
- ⌨️ Press "Enter" to add tasks quickly
- 🚫 Prevent duplicate tasks

---

## 🛠️ Built With

- HTML5
- CSS3 (optional styling)
- JavaScript (Vanilla JS)
- Browser Local Storage API

---

## ⚙️ How It Works

- Tasks are stored in a JavaScript array
- Every change updates the UI using `renderTasks()`
- Data is saved in `localStorage` for persistence
- User interactions trigger DOM updates in real-time

---

## 🧠 Key Concepts Practiced

- DOM manipulation
- Event handling (click, keydown)
- Array methods (`push`, `splice`, `some`, `forEach`)
- JSON and localStorage
- Basic CRUD operations
- UI rendering logic
